FAQ:

1.how to start a monitor?

From option menu "Add server", you can input a url of your server based on HTTP/HTTPS protocal and interval minutes of monitor, optional input is "Verify Title" that's useful when your server set a error 404 or 500 page to check if response page title contains this value

2.how to stop a monitor?

In the main window, you can turn on or off switch of every server to start or stop its monitor service.

3.how to know my server up or down?

In the main window, you can see green or red light icon on the left of every server, obviously green means up and red means down.

4.what happen monitor my server down?

Once monitor server starts, you can quit software window, when your server is down, the status panel of your phone will get a notification with default sound, vibrate, light.

5.Could I review the past log?

Yes. Click the button "History" of every server in the main window to review all of hitory logs.

本软件专为监控Web服务器运行状态,如果你也像我一样需要时时刻刻了解服务器是否正常运行，那么这就是为你准备的。

常见问题：

1.如何开始监控服务器？

从菜单中选择"添加服务器"，然后输入URL(必须是基于HTTP或HTTPS协议)，以及间隔时间，校验标题是可选项，当你的服务器配置了404或500错误页面的时候，就需要校验标题是否和正常页面一致

2.如何停止监控服务器？

在主界面中每个服务器都可以通过点击"开关"按钮来启动或者停止监控服务

3.如何知道我的服务器是否正常？

在主界面中每个服务器旁边都有绿色或红色两种图标,很显然绿色就是正常，红色就是不正常

4.监控服务发现我的服务器不正常时会做什么？

一旦监控服务启动，你就可以退出窗口界面，当你的服务器无法正常连接时，监控服务会在你的手机上方的状态栏发送一个通知，同时伴有默认的声音、震动和闪灯

5.我可以看到过去的监控记录吗？

可以.在主界面中点击每个服务器下面的"历史"按钮就可以查看历史记录
